<p>App is built around Clean architecture basics. Main resentation layer architecture pattern is MVVM.</p>
<p>Language: Kotlin</p>
<p>Used libraries:</p>
<ul>
  <li>Kotlin Coroutines (Flow is used for reactivity)</li>
  <li>Dagger(DI)</li>
  <li>Room</li>
  <li>Retrofit</li>
</ul>
